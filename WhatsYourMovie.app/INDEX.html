üöÄ Quick Start Guide

1 Install Node.js

Download and install from nodejs.org (choose LTS version)

Verify installation: Open Command Prompt and type node --version

2 Run Setup

Double-click setup.bat in this folder to install all dependencies

This will take 2-3 minutes and only needs to be done once.

3 (Optional) Configure TMDB API

For 100+ movies, get a free API key from TMDB

Create backend/.env file and add your key (see ENV_SETUP.txt)

OR skip this and use the 10 sample movies included!

4 Start the App

Double-click start.bat

Wait 30 seconds for both servers to start

Your browser will automatically open to http://localhost:3000

5 Create Account & Enjoy!

Sign up with any credentials (email can be fake for local use)

Click "Use Sample Movies" for instant testing

Select your Top 10 favorite movies

Start swiping and get personalized recommendations!
